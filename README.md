# Student Attendance Management System

A web-based Student Attendance Management System developed using PHP, MySQL, HTML, and CSS.

## Features

- Admin Login with Session Security
- Add Student
- View Students
- Edit Student
- Delete Student
- Mark Attendance
- Attendance Report
- Dashboard with Live Counts
- Clean UI Design

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
